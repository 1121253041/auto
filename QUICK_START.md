# ⚡ 5分钟快速上手

## 🎯 最简单的使用方式

### 步骤1: 启动项目（1分钟）

```bash
# 下载项目
git clone https://github.com/1121253041/auto.git
cd auto

# 安装并启动
npm install
npm run dev
```

### 步骤2: 打开应用（10秒）

浏览器访问：**http://localhost:3000**

### 步骤3: 添加第一个网页（30秒）

1. 点击右上角 **"添加网页"** 按钮
2. 输入：
   - URL: `https://example.com`
   - 标题: `示例网站`
3. 点击 **"添加"**

🎉 恭喜！你已经成功添加了第一个网页！

## 🔌 进阶：使用CORS代理

### 启动代理服务器（2分钟）

**新开一个终端窗口：**

```bash
cd proxy
npm install
npm start
```

看到这个就成功了：
```
🚀 CORS Proxy Server running on http://localhost:3001
```

### 使用代理加载跨域网站（30秒）

1. 点击 **"添加网页"**
2. 输入任何网站URL
3. **勾选"使用代理"**
4. 点击添加

现在可以加载任何网站了！🎉

## 🎨 体验全屏模式（10秒）

1. 点击任意网页右上角的 **⤢** 按钮
2. 该网页会全屏显示
3. 再次点击 **⤓** 退出全屏

## 🧩 安装浏览器扩展（可选，2分钟）

### Chrome浏览器

1. 地址栏输入：`chrome://extensions/`
2. 右上角打开 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择项目的 `extension` 文件夹

### 使用扩展

1. 点击浏览器工具栏的扩展图标
2. 打开 **"启用扩展"** 和 **"同步输入到iframe"**
3. 在任意iframe的输入框输入，会自动同步到其他iframe
