## 手动测试建议

由于Playwright浏览器下载较慢，建议进行以下手动测试：

### 浏览器测试步骤

1. **打开应用**
   ```
   http://localhost:3000
   ```

2. **测试添加网页**
   - 点击"添加网页"按钮
   - 输入URL: `https://example.com`
   - 输入标题: `测试网站`
   - 点击添加
   - 验证iframe出现

3. **测试全屏**
   - 点击全屏按钮（⤢）
   - 验证iframe全屏显示
   - 点击退出全屏（⤓）
   - 验证恢复网格布局

4. **测试插件面板**
   - 点击右上角"🔌 插件"按钮
   - 验证显示"Web Monitor"插件
   - 验证版本号"v1.0.0"
   - 关闭面板

5. **测试响应式**
   - 打开开发者工具(F12)
   - 切换到设备模拟模式
   - 测试不同设备尺寸

6. **测试持久化**
   - 添加几个网页
   - 刷新页面(F5)
   - 验证配置保留
